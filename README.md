## ЁЯФ╣Overview

Devan is a Sanskrit-inspired programming language that allows you to write logical code in the sacred style of Sanskrit. It supports both interpreting and compiling into modern languages like Python and PHP. Code files use the .Om extension and are translated using a custom Sanskrit-to-code mapping system.


---

ЁЯУж Key Features

ЁЯза Interpreter for .Om files тАУ Run directly via Python

ЁЯЫа Compiler тАУ Convert .Om to .py or .php seamlessly

ЁЯУЬ Sanskrit-to-Code Mapping тАУ Powered by a customizable JSON-based structure

ЁЯФД Auto AI Translation тАУ Translate English input to Sanskrit using offline Indic NLP models

ЁЯФР Local File Encryption тАУ Auto-encrypt original and translated files for privacy

ЁЯУж One-Line Installer тАУ Automatically install required Python packages

ЁЯТ╗ CLI Tool тАУ Simple commands like devan run and devan compile

ЁЯз░ PHP Runtime Support тАУ Execute .Om code in PHP using built-in executor

ЁЯз▒ Cross-Platform тАУ Works on Windows, macOS, Linux, and Ubuntu
